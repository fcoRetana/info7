    Modelado de Datos

    Es modelar el aspecto estático del sistema, es decir, las estructuras de persistencia de datos y las relaciones que existen entre ellas.

    Modelo Entidad-Relación

    El modelo entidad-relación ilustra las entidades como personas, objetos o conceptos que se relacionan entre sí dentro de un sistema. Se utilizan para diseñar o depurar bases de datos relacionales.

    Justificación.
    CMMI en Nivel 2-TS 2.1 menciona los modelos de entidad-relación dentro de las técnicas y métodos de diseño efectivos.
